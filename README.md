# 342. Power of Four

Given an integer n, return true if it is a power of four. Otherwise, return false. </br>

An integer n is a power of four, if there exists an integer x such that n == 4x. </br>

## Example 1:

Input: n = 16 </br>
Output: true </br>

## Example 2:

Input: n = 5 </br>
Output: false </br>

## Example 3:

Input: n = 1 </br>
Output: true </br>

## Constraints:

-2^31 <= n <= 2^31 - 1 </br>
